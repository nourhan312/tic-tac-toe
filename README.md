# Tic Tac Toe

A simple and engaging Flutter project.

## Overview

Tic Tac Toe, also known as X-O, is a two-player game where players take turns placing 'X' and 'O' on a 3x3 grid. The objective is to be the first to align three marks in a rowâ€”horizontally, vertically, or diagonally.

## Features
- **Intuitive Interface**: Easy to navigate and play.
- **Match History**: Automatically saves past game records.
- **Reset Button**: Clear previous records and start fresh.

## Screens
- **Splash Screen**: Initial loading screen.
- **Home Screen**: Start a new game or view match history.
- **Game Screen**: Play the Tic Tac Toe game.

## Tools & Packages
- **Lottie Animation**
- **Shared Preferences**
- **Cubit for State Management**
- **Google Fonts**
